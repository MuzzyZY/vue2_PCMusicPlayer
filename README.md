# vue_muisc_player

# 项目组成

由vue2+vue-router-vuex-element-ui制作的基于网易云api接口制作的网页端音乐播放器 

这次相对于我github库中的手机端音乐播放器增加了扫码登录功能 以及渲染出用户歌单的功能

## Project setup

```
pnpm install / npm install
```
